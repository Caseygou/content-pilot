# ContentPilot

ContentPilot is a full-stack content management dashboard for article publishing, media uploads, authentication, and analytics.

## Tech Stack

### Frontend

- React
- Vite
- TypeScript
- Ant Design
- React Router
- TanStack Query
- Zustand
- Axios
- ECharts

### Backend

- NestJS
- TypeScript
- Prisma
- MySQL
- JWT authentication
- Swagger/OpenAPI

## Project Structure

```txt
content-pilot/
  apps/
    web/        Frontend app
    api/        Backend API
  packages/
    shared/     Shared types and utilities
  README.md
```
